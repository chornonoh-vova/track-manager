# Track Manager

## Requirements

Setup the Node version

```bash
nvm use
```

Install the dependencies

```bash
npm Install
```

## Development

Start a project

```bash
npm start
```

## Main tasks

- [x] Create a Track
- [x] Edit Track Metadata
- [x] Upload a Track file
- [x] Delete a Track
- [x] List View with Pagination, Sorting and Filtering
